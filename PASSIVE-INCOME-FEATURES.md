## Mobile Deep Link Behavior

### iOS
```
1. User clicks "Open App"
2. Attempts: cashapp://cash.app/$username
3. If installed: Opens Cash App
4. If not: Falls back to https://cash.app/$username
5. Safari opens with payment page
```

### Android
```
1. User clicks "Open App"
2. Attempts: cashapp://cash.app/$username
3. If installed: Opens Cash App
4. If not: Falls back to https://cash.app/$username
5. Chrome opens with payment page
```

### Desktop
```
1. User clicks "Open App"
2. Immediately opens: https://cash.app/$username
3. Opens in new tab
4. No app installation attempt
```
