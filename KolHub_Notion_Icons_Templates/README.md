# Notion Templates & Icons – Setup Guide

**How to use**  
1. Import this folder into Notion (drag-drop).  
2. Open each page in `/templates`. Click the ⋯ menu → **Turn into template** (or duplicate, then set as template in the database).  
3. Assign icons and covers from `/icons` and `/covers`.  
4. For databases, add a **Template button** and choose one of these pages as the default content.

**Suggested icon → page mapping**  
- Dashboard → `icons/dashboard.svg` + cover `covers/goth-dark-1.png`  
- Health & Care → `icons/health.svg`  
- Support & Scripts → `icons/support.svg`  
- Wardrobe & Outfits → `icons/wardrobe.svg`  
- Ideas Library → `icons/ideas.svg`  
- Finances → `icons/finances.svg`  
- Entertainment → `icons/entertainment.svg`  
- ChronoMuse → `icons/chrono.svg`  
- Community → `icons/community.svg`  
- Settings & Automations → `icons/settings.svg`  

Tip: After importing, create **Template Buttons** inside Databases: `+ New` → choose your template as default for new entries.
