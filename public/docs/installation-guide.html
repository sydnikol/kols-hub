KOL Hub Installation Guide

Version 4.2.0 - Complete installation instructions for all platforms

Android Installation

Method 1: Direct APK Download (Recommended)

  1. Download the APK file from the Download Page
  2. Open your device's Settings
  3. Go to Security or Privacy
  4. Enable "Install from Unknown Sources" or "Allow installation from this source"
  5. Open the downloaded APK file
  6. Tap Install when prompted
  7. Once installed, tap Open to launch KOL Hub

Security Note

Android may show a warning about installing apps from unknown sources. This is normal for apps not from the Play Store. KOL Hub is safe to install - your data stays on your device.

Method 2: Web App (PWA)

  1. Visit kolhub.netlify.app in Chrome
  2. Tap the menu (three dots) in the top right
  3. Select "Add to Home Screen"
  4. Tap Add to confirm
  5. KOL Hub icon will appear on your home screen

iOS Installation (Web App)

KOL Hub works great on iOS as a Progressive Web App (PWA):

  1. Open Safari on your iPhone or iPad
  2. Visit kolhub.netlify.app
  3. Tap the Share button (square with arrow)
  4. Scroll down and tap "Add to Home Screen"
  5. Tap Add in the top right
  6. KOL Hub will appear on your home screen like a native app

PWA Benefits

The web app works offline, syncs your data, and receives automatic updates. No app store required!

Desktop Installation

Method 1: Web App (Recommended)

  1. Visit kolhub.netlify.app in Chrome or Edge
  2. Click the install icon in the address bar (or menu > "Install KOL Hub")
  3. Click Install to add to your applications
  4. KOL Hub will open in its own window and appear in your Start Menu/Applications

Method 2: Use in Browser

Simply bookmark kolhub.netlify.app and access it anytime from your browser.

Troubleshooting

Android: "App not installed" error

  • Make sure you have enough storage space
  • Uninstall any previous version first
  • Check that "Unknown Sources" is enabled
  • Try downloading the APK again

iOS: App won't add to Home Screen

  • Make sure you're using Safari (not Chrome or other browsers)
  • Try closing Safari and reopening the page
  • Check if you have restrictions enabled in Settings

Data not syncing

  • Check your internet connection
  • Make sure you're signed in
  • Try refreshing the app
  • Your data is stored locally and will sync when online

System Requirements

  • Android: Version 7.0+ (Nougat), 100MB storage, 2GB RAM
  • iOS: iOS 13+ with Safari
  • Desktop: Any modern browser (Chrome, Edge, Firefox, Safari)
